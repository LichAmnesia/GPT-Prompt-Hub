# Prompt Hub — 222 Battle-Tested Prompts for Claude, GPT, Gemini

> Author: Shen Huang

Not another "act as X" dump. Each prompt is a structured brief: **Role / Objective / Inputs / Output / Constraints / Example**. Designed for 2026-class models (Claude 4.x, GPT-5, Gemini 3) — not for 2023 ChatGPT.

## Who this is for

- Solo founders who ship alone and use LLMs as a full-stack team
- Engineers who want production-grade prompts, not chat screenshots
- PMs and operators writing specs, reviews, and strategy docs
- Marketers and SEOs running ads, content, and growth with AI leverage
- Researchers and students structuring reading, writing, and defenses

## How to use

- Each `.md` file follows the same structure: **Role / Objective / Inputs / Output / Constraints / Example**
- Designed for Claude 4.x, GPT-5, and Gemini 3 — `model: any` means it works across all three
- Drop the prompt into your chat, replace `${variables}` in the inputs block, go
- Chain with `lich-skills` for automated workflows if you use Claude Code or Codex

## Why this hub (and not another "act as X" list)

Most prompt collections are 2023-style lists of "Act as X" one-liners — fine for ChatGPT 3.5, noise for modern models. This hub is the opposite: **curated, structured, and small enough to know every entry.** Each prompt is a full brief with inputs, output contract, and constraints — upgraded for 2026-class reasoning models that reward structure. 222 prompts, sorted into 10 real categories — not 500+ scraped one-liners with duplicates.

If you want a dump, use the awesome list. If you want prompts that ship, use this hub.
